### Solution : 
We can solve this in 2 ways : 

The One - Liner Method : GO to the "inhere" directory and run : 

" find . -type f -size 1033c | xargs file | grep "ASCII text" "

Explanation : 

How this works:

   1. find . -type f -size 1033c: Locates every file (-type f) in the current directory (.) that is exactly 1033 bytes (1033c).
   2. | xargs file: Takes that list of files and runs the file command on each one to see what's inside (e.g., "data", "executable", or "ASCII text").
   3. | grep "ASCII text": Filters the results to show you only the one that is human-readable text.
