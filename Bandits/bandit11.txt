### Solution :

The password is stored in a file which has content encrpyted in ROT13 indirectly. 

To solve this we use the following command : "tr 'A-Za-z' 'N-ZA-Mn-za-m' < data.txt"

Here : 

tr: The command name. Means "translate"

'A-Za-z' (SET1): This is the "Search" set. It defines what characters the command should 
		 look for. A-Z covers all uppercase, and a-z covers all lowercase.

'N-ZA-Mn-za-m' (SET2): This is the "Replace" set. It defines what to swap the characters 
		       with.

< yourfile.txt: This tells the shell to take the contents of your file and "pipe" them 
		into the tr command.
