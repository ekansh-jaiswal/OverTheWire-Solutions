### Pass : FGUW5ilLVJrxX9kMYMmlN4MgbpfMiqey

### Solution : 

The pass is hidden inside data.txt which contains a base64 text.

We use the following command : "echo $(cat<data.txt) | base64 -d"

Output : 

bandit10@bandit:~$ echo $(cat<data.txt) | base64 -d
The password is dtR173fZKb0RRsDFSGsg2RWnpNVj3qRr
bandit10@bandit:~$

Working : We pipe the output of data.txt to base64 function and use the "-d" flag which
	  means to decrypt.
