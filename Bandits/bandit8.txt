### Solution :
To find a line that occurs exactly once in a file where every other line repeats, we combine the
sort and uniq commands.
The Magic Command:

sort data.txt | uniq -u

Why you need both:

    sort data.txt: The uniq command is a bit "lazy"—it only compares lines that are right next to each other. By sorting the file first, 
    we group all the identical lines together so uniq can spot them.
    uniq -u: The -u flag stands for "unique." Instead of just removing duplicates, it tells the computer: "Only show me the lines that had no partners at all."

### Output : 

bandit8@bandit:~$ sort data.txt | uniq -u
