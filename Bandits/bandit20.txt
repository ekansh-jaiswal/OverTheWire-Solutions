### SETTING_UP_TMUX : 

1. Run tmux command on the bandit machine.
2. Now press Ctrl+B twice to use tmux on bandit and use Ctrl+B once to use tmux on host.
3. Use Ctrl+B twice + Shift+5 to create vertically seperated tabs.
4. Use Ctrl+B twice + [ARR_KEY] to switch tabs.
5. Now we have 2 tmux tabs to work with!!

### THE_ACTUAL_SOLUTION : 

Setup a custom listening port, say 44444. Now run ./suconnect 44444. 
The port will silently connect without any message. This can be noticed when we observe
blank response on both the listener and the running terminal.
Now on the listening server input the bandit20 password as response. 
The suconnect binary would read and recieve this password and respond with password
for bandit21 as follows :
