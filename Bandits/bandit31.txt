Key Takeaways
Concept	Learning Point
Git Tags	Passwords/data can be hidden in tags, which are separate from branch history.
.gitignore	If git add isn't working, check for ignore rules; use -f to bypass them.
Pre-receive Hooks	Remote servers can run scripts to validate our code before allowing a push.
Git Reset	When local history gets messy or rejected, reset --hard is the best way to sync back to a "known good" state.
