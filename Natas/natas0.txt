# Solution : 

We can either use the browser or we can use curl. Here we will use curl. The same can bwe also done with browser as well. 
Here we first we will discuss each command as follows : 
1. curl -I http://natas0.natas.labs.overthewire.org

THis will give us the header flags when calling the website.

2. curl http://natas0.natas.labs.overthewire.org 

THis would simply give us the website. But since the request is incomplete we would get a failed response which can be observed in the bash transcript.

3. curl -u [USR]:[PASS] http://natas0.natas.labs.overthewire.org

Here we pass creds in the basic sign in box which was provided to us. And when we get logged in. We can see the contents of the website. 
And in the content , the pass for Natas1 is hidden as a comment.
