Here are the few things which we can notice from the given code :
    1. XOR encryption is being used to encrypt the cookies. This can be easily broken since
    XOR uses Symmetric Encryption. i.e.
    A XOR B = C <---> C XOR B = a
    or
    TEXT XOR KEY = CIPHER <-------> TEXT XOR CIPHER = KEY

    We use this logic and use the following php code to break the key :
