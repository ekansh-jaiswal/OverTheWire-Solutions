# Pass : natas9:ZE1ck82lmdGIoErlhQgWND6j2Wzz6b6t

# Solution : 

If we observe the source-code behind the input taking machine we get the follows : 

'''
<?
$key = "";

if(array_key_exists("needle", $_REQUEST)) {
    $key = $_REQUEST["needle"];
}

if($key != "") {
    passthru("grep -i $key dictionary.txt");
}
?>
'''

Here based on this code, we realize that we need to escape the shell in order to execute more commands.
Since the $key is directly being used instead of being converted into a string. We can escape this by using ";" character.
and the password for natas10 is located at /etc/natas_webpass/natas10.
