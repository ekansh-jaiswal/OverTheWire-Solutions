## Overview

This repository contains personal notes and solutions for OverTheWire wargames.

It is intended to serve as:

- a learning log
- a revision resource
- a collection of command-line and security notes
- a record of gradual progress through different wargames

Unlike a finished writeup repository, this one is **growing progressively** as more levels are solved.
